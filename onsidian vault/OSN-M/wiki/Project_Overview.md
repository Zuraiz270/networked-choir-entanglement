# SNA-OSN-M — Project Overview

Top-level synthesis for the Summer-2026 seminar *"Projekt zu Online Social Networks"* (SNA-OSN-M) at Otto-Friedrich-Universität Bamberg, run jointly with Uni zu Köln and Hochschule Luzern. **Rubedo tier** — aggregates all lower-stage pages; deepens as sources accumulate.

---

## What the seminar is

SNA-OSN-M studies online social networks through the **COINs** (Collaborative Innovation Networks) lens popularized by [[Peter_Gloor]]. The Summer-2026 edition is anchored by Gloor's book *Cybernetic Alchemy: From Paracelsus to Plant Sensors*, which supplies both the curriculum (16 chapters presented by student teams) and the structural metaphor for this wiki (Nigredo → Albedo → Citrinitas → Rubedo).

*Source*: [[info_txt]], [[seminar_signup]].

## Measurement Apparatus — SocialCompass Tools

All students completed four tools before the block course (see [[socialcompass_tools]]):

- **Symbiont Analyzer** — collaboration archetype (Bee/Ant/Butterfly/Capybara/Leech) from WhatsApp export.
- **SC Chat Analyzer** — network role + response latency from WhatsApp export.
- **Perceptiface** — dominant facial emotion + FFI Big Five from smartphone video.
- **Beecome** — FFI Big Five from behavioral swipe game.

Cohort finding: **Capybara dominates** Symbiont; **Tree Hugger** dominates SC Chat network position; **Neutral/Sad** dominates Perceptiface. (*Source*: [[seminar_signup]].)

## The Cybernetic Alchemy metaphor

The wiki's four-stage schema (Nigredo → Albedo → Citrinitas → Rubedo) comes from Gloor's book. Chapter 14 ("Data as Prima Materia", Part V: The Convergence) applies this same pipeline to modern machine learning across plant / animal / human layers. The chapter's core claim: the transformation is structural, not metaphorical. *Source*: [[chapter_14_presentation]].
