# Project Overview

> [!NOTE]
> This project is currently being redefined. 

## 🎯 Objective
The primary goal is to investigate **Networked Choir Entanglement**—measuring and enhancing team flow and synchronization in remote, low-latency musical environments.

## 🔬 Core Focus Areas
1. **Networked Music Performance (NMP)**: Leveraging JackTrip and other low-latency frameworks.
2. **Audio-Visual Synchronization**: Analyzing the effects of latency on perception and performance.
3. **Team Entanglement**: Measuring group flow and synchronization failures.

## 📂 Research Questions
- *How does network latency specifically impact the 'entanglement' of a chamber choir?*
- *Can multi-modal metrics (audio/visual) predict synchronization failure in NMP?*

## 🛠️ Open Questions
- [ ] Which specific synchronization metrics from the new literature are most applicable?
- [ ] How will we calibrate for the Bamberg/Cologne/HSLU network conditions?
